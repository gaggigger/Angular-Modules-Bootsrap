Welcome on Bootstrap for Angular modules !

This bootstrap creates full-stack project (Express/Angular), a watcher development server for HTML, JS, and CSS files.
If you read this lines, it's that the development server is ready for auto-compilation JS and SCSS files

An authenticate module was recently added, it based on Json Web Token (JWT). Simple middleware for control users, modo and admin was added too.

Introduction

Use "gulp" command for started an watch builder who generate main.js and styles.css files.
If you have need minified versions, just use the command "gulp prod" for generate it.

Check the about page for more technical informations on this bootstrap.
